# ComfyUI nodes to use QWQ32B

https://huggingface.co/Qwen/QwQ-32B


## Install

- 推荐使用管理器 ComfyUI Manager 安装（On the Way）

- 手动安装：
    1. `cd custom_nodes`
    2. `git clone https://github.com/leeguandong/ComfyUI_QWQ32B.git`
    3. 重启 ComfyUI
